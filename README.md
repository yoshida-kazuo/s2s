# S2S

## ディレクトリ構成
```
./
|-- src/
|   |-- controllers/
|   |   |-- homeController.ts
|   |   |-- userController.ts
|   |   |-- ...
|   |-- models/
|   |   |-- userModel.ts
|   |   |-- ...
|   |-- views/
|   |   |-- index.ejs
|   |   |-- user.ejs
|   |   |-- ...
|   |-- routes/
|   |   |-- homeRoutes.ts
|   |   |-- userRoutes.ts
|   |   |-- ...
|   |-- server.ts  (or app.ts)
|-- package.json
|-- tsconfig.json
```
